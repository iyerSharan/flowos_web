Privacy First

Built for people
who take privacy seriously.

Every single operation in FlowOS runs on your machine. Indexing. Embedding. LLM inference. Vector search. There is no server. No telemetry.

0
bytes ever sent to any server
Zero datacenter energy.
Inference runs on hardware you already own — no GPU farms, no cooling load, no carbon overhead per query.
🖥
Runs on your machine
FlowOS is a native desktop app for macOS and Windows. LLM inference runs via Ollama directly on your CPU or GPU. No internet connection required after setup.
🗄
Local vector database
All embeddings and document indexes are stored in ChromaDB on your local filesystem. The database lives in your home folder. You own it entirely.
🔑
No account required
FlowOS has no login, no user account, no subscription. Download and run. Your files never touch any authentication server or identity provider.
📦
Encrypted memory packs
When you export a .flowwrap bundle, it's encrypted before it touches disk. Only the machine that imports it can read the contents.
